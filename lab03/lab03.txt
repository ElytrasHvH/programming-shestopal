Лабораторна робота N3 Розробка лінійних програм 

Шестопал Дмитро Олексійович

Завдання: Підрахувати суму чисел у заданому діапазоні.
 Основна частина:
  - опис роботи основної функції: сумма усіх чисел у діапазоні визачається за формулою -1/2*(min-max-1)*(min+max)
  - перелік вхідніх даних:
  - a - мінімальне число, ціле яке число (int);
  - b - мінімальне число, ціле число (int);
 Дослідження результатів роботи програми
  - c - сумма чисел, ціле число (int);
 Код:
 int main()
{
        int a = 1; //first number
        int b = 22; //last number
        int c = (-0.5) * (a - b - 1) * (a + b); //calc formula

        return c;
}
  - при значеннях a = 1 та b = 22, c має дорівнювати 253
 - для підтвердження коректності роботи програми, зупинено відлагодник на строці з "return c"
 та введемо команду "print c", Після вводу команди отримали наступне:
(lldb) print c
(long double) $15 = 253
Результат співпав
Якщо змінити значення a або b до розрахунку с , то результат зміниться, але все-ще буде правильним щодо нових значень
Перевіремо: ставимо бреакпоінт на строці розрахунку с.
змінюємо a = -10 , b = 23, очікованна відповідь - 223
(lldb) expr a = -10
(int) $0 = -10
(lldb) expr b = 23
(int) $1 = 23
переходимо до return c;
(lldb) var c
(int) $2 = 221
Результат співпав з очікуванним
