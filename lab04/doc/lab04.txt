Завдання:  Дано дійсне число x. Для функції y(x), графік якої подано на рисунку нижче, обчислити значення y.
 Основна частина:  - опис роботи основної функції: подання вхідного аргументу та повернення значення функції з такими правилами:
 x < -1 -> y : -1/x
 -1 <= x <= 1 : x^2
 x > 1 = : 1
 перелік вхідних даних: 
 x - значення аргументу, може бути будь яким числом (float)
 дослідження результатів роботи:
 y - значення функції
 Код:
 int main()
{
        float x = 0.5f; // input var
        float y; // answer var
        if (x <= -1.0f) // check first condition
        {
                y = -1.0f / x;
        } else if (x<1.0f && x> - 1.0f) // check for second condition
        {
                y = x * x;
        } else // force value if 1 & 2 conditions didnt work 
        { 
                y = 1.0f;
        }

        return y;
}

 при значенні x = 0.5 - очікуванне значення = 0.25
 при значенні x = 2 - очікуванне значення = 1
 при значенні x = -2 - очікуванне значення = -0.5
 для підтвердження роботи програми ставимо бреакпоінт після надання x значення але до початку перевірок умов та змінюємо x на 0.5
 (lldb) expr x = 0.5f
(float) $0 = 0.5
переходимо до return y; та перевіряємо значення y
(lldb) var y
(float) y = 0.25
перезапускаємо програму та ставимо 2
переходимо до return y; та перевіряємо значення y
(lldb) var y
(float) y = 1
перезапускаємо програму та ставимо -2
(lldb) var y
(float) y = 0.5
Результати співпали з очікуванними. 
Якщо змінити x на таке значення що перевірка цього сегменту значень пройшла - через те що у останньому випадку нема перевірки на x>1 и зразу ставиться else y = 1.0f то виходить що у цьому випадку "y" завжди буде дорівнювати 1
